INSTALL.md

Install

Local setup is intentionally mock-first so the FastAPI app can run before real Webex credentials, token-manager wiring, or device access are available. 

python3 -m venv .venv
.venv/bin/python -m pip install -e ".[dev]"
.venv/bin/python -m uvicorn assistant_app.main:app --reload
  • Requires Python 3.12 or newer.
  • WEBEX_MOCK_MODE and DEVICE_MOCK_MODE default to true.
  • Real device access depends on the token manager sidecar and Webex cloud xAPI APIs.

USER_MANUAL.md

User Manual

Natural-language requests work through local debug routes today and through Webex messages in real mode, with approval cards handling most mutating actions.

  • Use POST /debug/messages for local testing and Webex messages for real ingress.
  • Read-only actions can run immediately; most mutating actions create approval requests by default.
  • The current admin page shows settings, provider data, policies, devices, approvals, audit records, and process-local stats.
